Version 0.1.0 (2011-02-17)

New Features

  • Initial Implementation! Please note, this is the first alpha version. It lacks both data persistence and networking capabilities.
  • Basic main window implemented placeholders for later graphics. Click boxes with labels to go to the relevant page.
  • Basic forum implementation completed. Posts unread by the current user will be highlighted.
  • Basic map implementation completed. Left click to select pins that have been dropped. Right click to drop a new pin.
  • Basic game creation started. Students can be added via XML file, and groups can be created/filled in the Game Creation window. As groups are added to the window, they will be added to the forum.
  • Basic dev tools window added. Select a student in the dev tools window and click "Set User" to set that user as the system's "current" user.

Bugs fixed
